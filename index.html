About Me

Biografi

Nama Saya Kiki Amalia sapitri saya seorang Mahasiswa semester 3 di Universitas Kuningan. Saya lahir dan dibesarkan di Kuningan. Saat ini, saya sedang menempuh pendidikan S1 di bidang Sistem Informasi, di mana saya belajar tentang berbagai aspek teknologi informasi, termasuk pengembangan perangkat lunak, manajemen database, dan analisis sistem. Saya percaya bahwa teknologi memiliki potensi yang besar untuk memecahkan masalah kompleks dan meningkatkan efisiensi dalam berbagai sektor.

Keterampilan & Tujuan karier

Saya memiliki keterampilan teknis seperti dalam menggunakan SQL untuk manajemen data. Pengembangan Web seperti membangun aplikasi web responsif menggunakan HTML, CSS, PHP. Selain keterampilan teknis, saya juga memiliki kemampuan komunikasi yang baik dan mampu bekerja sama dalam tim. Saya percaya bahwa kolaborasi adalah kunci untuk mencapai hasil yang optimal. Saya bercita-cita untuk bekerja di perusahaan teknologi terkemuka di mana saya dapat terus belajar dan mengembangkan keterampilan saya.
